anaconda

Date Sun 23 November 2014 Tags python

anaconda is a python distribution for scientific computing. I used to use epd distribution but this one is better including tons of python packages built-in. The packages of numpy, scipy, ipython, numba, matplotlib are all included. The installation is also straight forward. I would like to recommend this distribution of python for scientists. It is totally free.
